### Commit Message Convention

- title: `type: content`
  - Add(파일추가)/Update(내용추가 및 변경)
  - 파일이름이 아닌 어떤 내용을 담는 파일인지 명시하기
  - 어떤 공부를 했는지 요약할 수 있는 내용
- body: 본문내용

  - 공부한 대상과 학습 이유를 밝히기

```
Add: angular working flow

```
