# TypeScript의 장점
## 정적 타입
```ts
function sum(a: number, b: number){
    return a + b;
}
```
- 인자에 타입을 명시함으로서 컴파일 단계에서 오류를 포착할 수 있다.
- `sum('x', 'y')` 호출 시 type 오류를 리턴한다.

## 도구의 지원
- IDE에서 타입 체크, 코드 어시스트, 리팩토링이 지원된다.

## 강력한 객체지향 프로그래밍 지원
- 인터페이스, 제네릭 등으로 크고 복잡한 프로젝트의 구현이 가능하다.
- 기존 객체 지향 프로그래머의 진입 장벽을 낮출 수 있다.
